# 🚌 Bus Timetable

港区周辺のバス時刻表をスマホで手軽に確認できるWebアプリです。

**URL:** https://atog2.github.io/bus-timetable/

## 機能

- **リアルタイム時計** — 1秒ごとに更新
- **次のバス表示** — 現在時刻から最も近いバスの発車時刻と残り分数を大きく表示
- **バス停タブ切り替え** — 6つのバス停をタブで素早く切り替え
- **曜日切り替え** — 平日 / 今日 / 土曜 / 日祝 を手動で切り替え可能。「今日」選択時は日付から自動判定
- **祝日自動判定** — 2026年の国民の祝日を内蔵し、「今日」モードで正確なダイヤを表示
- **ルートフィルター** — バス停ごとに路線単位で表示・非表示を切り替え可能
- **全便表示** — デフォルトは直近15件、ボタンで当日全便を表示
- **自動再描画** — 毎分0秒に合わせてリストを自動更新

## 時刻表データの更新方法

`index.html` 内の `const STOPS = [...]` ブロックを編集してください。

```javascript
{
  id: "shiba1",
  name: "芝一丁目",
  routes: [
    {
      id: "shiba-shimbashi",
      label: "芝ルート 新橋行",
      color: "#f97316",
      weekday:  ["08:33", "09:03", ...],  // 平日
      saturday: ["08:14", "08:44", ...],  // 土曜
      holiday:  ["08:14", "08:44", ...]   // 日祝
    }
  ]
}
```

編集後は以下を実行してデプロイ：

```powershell
git add index.html
git commit -m "fix: 時刻表を更新"
git push
```

---

## 祝日データの更新方法

`index.html` 内の `const HOLIDAYS_2026` に `"YYYY-MM-DD"` 形式で追加してください。
年をまたぐ場合は変数名も合わせて更新してください。

---

## 注意事項

- 時刻表データは作成時点（2026年5月）の情報です。ダイヤ改正後は手動での更新が必要です
- 田町駅前（芝ルート）は2026年4月ダイヤ改正が未反映の可能性があります
- 公式時刻表は各バス会社のサイトでご確認ください
  - ちいばす：https://www.fujiexpress.co.jp/chiibus/timetable/
  - 都営バス：https://www.kotsu.metro.tokyo.jp/bus/
